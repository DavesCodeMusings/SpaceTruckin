About

SpaceTruckin' was created out of a desire for learning during COVID-19 quarantine. It's a web-based game that is built in, small easy to tackle stages. The game itself is similar to an old Apple II game I used to play, called Taipan! Inspiration for the title comes from a classic tune by Deep Purple.

I've broken the game into stages as I've been building and documented each one with a README.txt file explaining what's going on. My hope is to add some video instructions once it's all done.

Who is it for?

SpaceTruckin' is for anybody.

  • If you're bored and you need a distraction, play the latest working demo of the game.
  • If you're interested in coding, download the source code and follow along in the development process.
  • If you're a teacher, use it to show your class how a simple game can be built from the ground up, with examples.

What will you learn?

If you decide to follow along with the development process of the game, here are a few things you will learn about along the way:

  • How to build web pages with HTML, CSS and JavaScript.
  • Ways to enhance your pages with images and custom fonts.
  • How to make your web pages work on mobile platforms like phones and tablets.
  • Using a rapid and iterative development process gets results quickly.

What do you need to get started?

Space Truckin' requires nothing more than a plain old text editor. Honest. Everything can be written in Notepad if you want. I've used Microsoft's free to download tool, Visual Studio Code, but alternatives are Notepad++ and a host of other code editors available on the internet.

Unfortunately, I am not familiar with code editor offerings on MacOS or ChromeOS, but I hope to add them soon.
